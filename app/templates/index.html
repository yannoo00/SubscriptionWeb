품앗이 서비스에 오신 것을 환영합니다!

프로젝트에 참여하고, 서로의 멘토가 되어주는 품앗이 서비스입니다.

프로젝트 목록 보기

프로젝트 참여 가이드라인

📌 프로젝트 진행 요약

  • 핵심 기능을 구현하는데 필요한 함수 순서를 나열해서 그 순서대로 기능을 우선적으로 구현합니다.
  • 기능 추가 시마다 해당 기능 구현에 필요한 함수 호출 순서를 나열하여 개발합니다.
  • 각 기능마다 함수 호출순서표를 노션에 기록합니다. (필수사항)

📌 중요 부분

  • 기획설명 → 구조설계 → 순서도 → 구현 순으로 진행합니다.
  • 순서도는 매일 업무 시 더욱 세부적인 동작 순서로 발전시켜 나갑니다.
  • 실행방법과 실행코드를 노션에 반드시 기록해 주세요.

📌 프로토타입 중심의 개발

  • 첫 1-2일 안에 프로토타입 실행 버전을 만들어 주세요!
  • 핵심 기능을 먼저 구현하고 부가적인 기능을 추가해 나갑니다.
  • 빠른 개발 주기를 유지하여 프로토타입을 반복적으로 개선합니다.
  • 사용자 중심의 설계를 통해 지속적으로 프로토타입을 개선합니다.

📌 작업 현황 작성 방법

  1. 작업한 날짜와 그날 한 일에 대한 제목을 적습니다.
  2. 토글 또는 연구일지 백링크를 통해 상세 설명을 작성합니다.
  3. 상세 설명에는 작업 설명, 코드, 실행 결과 화면 등을 포함합니다.
